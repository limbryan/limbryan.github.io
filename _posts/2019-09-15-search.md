## Search!

In a previous post about programming a Sudoku solver, a DFS (Depth First Search was used). DFS is pretty basic and primitive - almost a brute force search. There are other search algorithms out there too. This was one the few topics in my 6.034 (Artificial Intelligence) class at MIT. I like how this class focuses alot of not just the basics but also what is important and how we think about intelligence.
Also, coming from a mechanical engineering backgroud, I like how I am finally getting a more formal class/introduction into these more computer science based things. Anywho, here is some general notes I wrote about search and the various types of search.


### Blind Search 
1. Brute Force Search ("British Musuem")
2. DFS - Depth First Search 
3. BFS - Breadth First Search 

### Informed Search/Heuristic 
1. Hill Climbing
2. BEAM Search 
3. Best-first Search

### Optimal Search 
1. Branch and Bound 
2. Branch and Bound with extended set
3. Branch and Bound with heuristic
4. A* (Branch and Bound with heuristic and extended set)
